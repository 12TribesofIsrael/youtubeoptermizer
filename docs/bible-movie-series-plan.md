## Episode Production Workflow

Each episode follows this pipeline:

```
1. Scripture Selection → Pull relevant KJV chapters
2. Script Writing → Claude AI + manual review (narration + scene descriptions)
3. Visual Generation → Kling AI prompts (follow character depiction rules)
4. Text Cards → ChatGPT/DALL-E for title card + closing scripture cards
5. Audio → Narration (AI voice or manual recording)
6. Assembly → CapCut (visuals + narration + text cards + music)
7. Thumbnail → DALL-E 3 via dashboard tool (gold text + cinematic scene)
8. Upload → YouTube API via dashboard
9. Metadata → Optimized title, description, tags via dashboard
10. Cross-post → Short clip to IG/TikTok/X/FB via Repurpose.io
```

### Character Depiction Rules (MANDATORY for every visual)

All Israelite characters MUST be depicted as melanated Black/African American:
- "dark-brown to deep-brown skinned man/woman"
- "melanated African American complexion"
- "wool-textured / coiled / tightly curled hair"
- "NO Caucasian, European, or light-skinned depictions"

### Text Rules for Kling AI

- NEVER include text in Kling prompts (Kling cannot render text)
- Narration carries all scripture — viewer hears it, Kling shows visuals
- Only generate text cards in ChatGPT/DALL-E (gold serif on black background)
