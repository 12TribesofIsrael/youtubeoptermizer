What This Tool Does

The YouTube Optimizer is a full-stack channel management platform purpose-built for the AI Bible Gospels channel (@AIBIBLEGOSPELS). It replaces manual YouTube Studio workflows and scattered Python scripts with a single web interface that handles everything — from bulk video editing and playlist management to AI-powered social media audits and cross-platform posting.

The channel focuses on the 12 Tribes of Israel, Hebrew Israelite identity, and biblical prophecy through AI-generated cinematic content. This tool was built to scale the channel from ~5,900 subscribers toward 10K+ by automating optimization, content distribution, and audience growth workflows.

How to Run

# 1. Install dependencies
pip install -r requirements.txt

# 2. Set environment variables (create .env file)
ANTHROPIC_API_KEY=your-key          # For social media audits
META_ACCESS_TOKEN=your-token        # For Facebook posting
FACEBOOK_PAGE_ID=your-page-id      # For Facebook posting
TWITTER_API_KEY=your-key            # For Twitter posting
TWITTER_API_SECRET=your-secret
TWITTER_ACCESS_TOKEN=your-token
TWITTER_ACCESS_SECRET=your-secret
OPENAI_API_KEY=your-key             # For scripture card generation

# 3. Set up YouTube OAuth (first time only)
# Place credentials.json from Google Cloud Console in project root
python scripts/connect.py

# 4. Start the dashboard
python app.py

# Visit http://localhost:8080
