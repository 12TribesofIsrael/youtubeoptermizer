**Last Updated:** April 1, 2026
**Purpose:** Paste these into Repurpose.io caption settings for each platform. Every new video will auto-post with viral captions.

## How to Set Up in Repurpose.io

1. Go to your Repurpose workflow for each platform
2. Click **Edit** → **Caption** section
3. Paste the template for that platform
4. Use Repurpose variables where noted: `{title}` = video title, `{url}` = link
